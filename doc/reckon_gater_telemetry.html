    View Source

    Telemetry handler for reckon-gater

    Provides logging handler for telemetry events and utilities for attaching/detaching handlers.

    Usage

    Attach the default logger handler: ok = reckon_gater_telemetry:attach_default_handler().

    Attach a custom handler: ok = reckon_gater_telemetry:attach(my_handler, fun my_module:handle/4, #{}).

    Emit an event: reckon_gater_telemetry:emit(?GATER_REQUEST_STOP, #{duration => 1000}, #{store_id => my_store}).
